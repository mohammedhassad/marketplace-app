## Acknowledgements
<p>Built a full-stack web app to allows users to become sellers, who can manage multiple shops and add the products they want to sell in each shop. Users who visit the application will be able to search for and browse products they want to buy and add products to their shopping cart to place an order.</p>

<p><strong>Used:</strong> NodeJs, ExpressJs, MongoDB, Mongoose, ReactJs, Tailwindcss, GraphQL.</p>
